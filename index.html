Desky: Open Source Macropad

Featuring magnetic expansion connectors for modularity and ease of use.

What is Desky?

Desky is an innovative, customizable macropad with magnetic expansion connectors. You can easily add modules to expand Desky's functionality. Perfect for developers, creators, and anyone looking to streamline their workflow while learning programming and electronics.

Specifications

  • Magnetic I2C connectors for easy module expansion
  • Open-source hardware and software
  • Customizable key layouts and macros
  • Simple setup and configuration
  • Compact and portable design
  • Powered by the Adafruit KB2040

Installation & Usage

To get Desky and get started, follow these steps:

  1. Go to the GitHub repository and download the PCB files: GitHub repository.
  2. Go to PCBWay or a similar service to manufacture and assemble the Desky PCB.
  3. Connect Desky to your computer using the provided cable.
  4. Drag and drop the Desky .UF2 file to the drive that appears.
  5. Customize Desky's macros and key layout using the provided tools.
  6. Start using Desky to improve your workflow and learn fundamental programming concepts.
