Desky: Open Source Macropad

Featuring magnetic I2C connectors for modularity and ease of use.

What is Desky?

Desky is an innovative, customizable macropad with magnetic I2C connectors. It allows you to add modules to expand functionality with ease. Perfect for developers, creators, and anyone looking to streamline their workflow with physical macros and shortcuts.

Features

  • Magnetic I2C connectors for easy module expansion
  • Open-source hardware and software
  • Customizable key layouts and macros
  • Simple setup and configuration
  • Compact and portable design

Installation & Usage

To get started with Desky, follow these steps:

  1. Download the latest firmware and software from the GitHub repository.
  2. Connect Desky to your computer using the provided cable.
  3. Install the necessary drivers (if any).
  4. Customize the key layout and macros using the Desky configuration software.
  5. Start using your macropad and improve your productivity with customizable shortcuts!
